Mis Estudios


En general estoy estudiando los cursos ofrecidos por la página de Coderhouse.

También aprendiendo a través de videos tutoriales de diferentes plataformas, como youtube, o blogs de internet en general.

Además de leer no solo documentación sobre los lenguajes que estudio, si no también de diferentes libros sobre los temas cursados, como Python y actualmente Html y Css

Curso: Python

El curso de Python dictado en la plataforma de Coderhouse, se desarrollo en 25 Clases de 2hs cada una, más clases extras (afterclass) de una hora en la que repasabamos temas vistos y hasta vimos temas que no estaban en el programa. Arrancamos viendo el lenguaje puro de Python para terminar utilizando un Framework - Django - para realizar el backend de una página con tema a elección, con creación de usuarios y contraseñas, CRUD, creación de blog y programación orientada a objetos.

¿Qué es el Top 10?

Es el ranking de los mejores estudiantes de la camada. Para estar en el Top se deben sumar puntos a través de las asistencias, valoraciones, desafíos, entregas intermedias y el proyecto final, para acceder al mismo y obtener sus beneficios.⁠


En mi caso personal logre terminar el curso de python con una nota en el proyecto final de 10, y estar en el top como figura en el certificado.
